### CLI Implementation ✅
- Created `cli.py` with Click framework
- Added `click>=8.1.0` to dependencies
- Implemented commands:
  - `codecourt review <file>` — review a diff file
  - `codecourt review --stdin` — read diff from pipe
  - `codecourt review --repo <path>` — review git changes
  - `codecourt providers` — list available LLM providers
  - `codecourt parse` — debug diff parsing
- Features:
  - Provider selection (`--provider openai|anthropic|ollama`)
  - Model selection (`--model gpt-4o`)
  - Output formats (`--format rich|json|markdown`)
  - Parallel/sequential execution toggle
  - Verbose mode for debugging
  - Rich terminal output with tables and colors
  - Proper exit codes (0=ok, 1=changes requested, 2=critical issues)
